# 📸 Batch BG Remover Pro (Workflow Edition)

[![GitHub stars](https://img.shields.io/github/stars/gohard-lab/batch_bg_remover_pro?style=social)](https://github.com/gohard-lab/batch_bg_remover_pro)

**The ultimate AI-powered batch background removal tool.** This project has evolved from real-world user feedback to maximize workflow efficiency for e-commerce sellers, photographers, and developers. It goes beyond simple AI background removal by focusing on professional utility and bulk processing stability.

---

## ✨ Key Features (v2.0 Update)

* **📂 Mirror Folder Structure**: Automatically replicates your original directory hierarchy in the output folder, saving you hours of file organization.
* **🎨 Custom Canvas & Background**: Supports custom background colors (HEX codes) and fixed canvas dimensions (e.g., 1000x1000) for instant marketplace compatibility.
* **⚠️ Smart Error Isolation**: Any files with uncertain cutouts or processing errors are automatically moved to a `_review_needed` folder for easy manual inspection.
* **📊 Performance & Accuracy Logs**: Generates a detailed `process_stats.csv` report for every batch, tracking processing time, success status, and hardware efficiency.

---

## 🚀 Quick Start

This project uses **`uv`**, the modern standard for Python package management, to ensure a seamless and fast setup.

```bash
# Clone the repository
git clone [https://github.com/gohard-lab/batch_bg_remover_pro.git](https://github.com/gohard-lab/batch_bg_remover_pro.git)
cd batch_bg_remover_pro

# Install dependencies and run
uv sync
uv run main.py
```
